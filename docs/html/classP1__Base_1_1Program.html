static int P1_Base.Program.FindLeastCommonMultiple ( List< int > firstPrimeFactors,
List< int > secondPrimeFactors )
staticprivate

Calculates the Least Common Multiple (LCM) of two numbers using their prime factorizations. The LCM is computed by taking each prime factor to the highest power in which it occurs in either number.

Parameters
firstPrimeFactorsList of prime factors of the first number number
secondPrimeFactorsList of prime factors of the second number
Returns
The least common multiple of the two numbers represented by their prime factors. For example, for inputs [2,2,3] and [2,3,3], representing 12 and 18, returns 36.
